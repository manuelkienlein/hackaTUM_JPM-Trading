# hackaTUM_JPM-Trading
Stock Exchange Platform that allows you to trade various stocks via API. Built with &lt;3 in Munich at hackaTUM 2022.

# Example api routes
- http://127.0.0.1:8000/registration
- http://127.0.0.1:8000/accounts/login/
- http://127.0.0.1:8000/accounts

- http://127.0.0.1:8000/api/users/1/stocks/buy?stock=MSFT&quantity=10&price=999
- http://127.0.0.1:8000/api/users/1/stocks/sell?stock=MSFT&quantity=10&price=999
- http://127.0.0.1:8000/api/users/1/orders
- http://127.0.0.1:8000/api/users/1
- http://127.0.0.1:8000/api/orders
- http://127.0.0.1:8000/api/stocks/matching
